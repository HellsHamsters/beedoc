# Введение в модули

BeeGo развивается в стиле LEGO, то есть как модульная система. Каждый модуль как один кирпичик и вы можете строить ваше приложение из тех блоков из которых хотите. Начиная с BeeGo 1.0 мы имеем вот эти модули:

- [Session Module](./session.md)
- [Cache Module](./cache.md)
- [Logs Module](./logs.md)
- [Httplib Module](./httplib.md)
- [Context Module](./context.md)
- [Toolbox Module](./toolbox.md)
- [Config Module](./config.md)
